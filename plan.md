Next.js 15(App Router)와 Tailwind CSS를 기반으로, 모바일에서 'iPhone 광고'처럼 웅장한 감성을 주는 '4000weesk' 프로젝트를 개발해줘.

[1. Route 1: 메인 홈 (/) - 브랜딩 및 온보딩]
- 목적: 강렬한 유튜브 쇼츠 후킹 및 사용자 정보 입력.
- 오프닝 시퀀스:
  1. 접속 시 칠흑 같은 화면에서 유튜브 쇼츠(9ltA6xvDscE)를 Full-screen(모바일 최적화)으로 자동 재생.
  2. 브라우저 정책을 위해 'Muted'로 시작하되, 중앙에 웅장한 'Unmute & Start' 버튼 배치.
  3. 로컬 스토리지를 확인해 이미 방문한 유저는 영상을 스킵하고 바로 입력 폼으로 진입.
- 입력 플로우:
  - 영상 종료/스킵 시, 영상이 위로 슬라이드되며 '입력 폼' 등장.
  - 필드: 
    * 한글 이름(username): UI 표시용 (예: "조정제").
    * 영문 ID(domain_key): URL 경로용 (`4000weesk.kro.kr/[domain_key]`). 한글 입력 시 자동 영문 변환 제안.
    * 생년월일: 모바일 네이티브 데이트 피커.
  - 완료 시 데이터를 Vercel KV(인터페이스)에 저장하고 `/[domain_key]`로 이동.

[2. Route 2: 개인 페이지 (/[domain_key]) - 인생 대시보드]
- 특징: 이 경로로 직접 접속 시 오프닝 영상은 생략하고 바로 시계 UI 노출.
- UI/UX:
  - 4,000개의 미세한 도트(Dot)가 모바일 화면을 가득 채운 '라이프 그리드' 시각화 (Canvas API 사용).
  - 이미 지나온 시간은 흐릿하게, 남은 시간은 선명하게, 이번 주 도트는 Red Pulse 효과.
  - 상단: "[한글이름]님의 4,000주" 타이포그래피.
  - 하단: 이번 주가 가기까지 남은 시간 실시간 카운트다운.
- 네비게이션 & 바이럴:
  - 좌측 상단 '4000WEESK' 로고 클릭 시 메인 홈(`/`)으로 이동.
  - 화면 하단에 "나도 인생 시계 만들기"라는 세련된 플로팅 CTA 버튼 배치 (제3자 방문객 유입용).

[3. 디자인 시스템 & 인터랙션 (Apple Style)]
- 테마: Jet Black(#000000), Grain Noise 텍스트, 포인트 컬러 Red(#FF3B30).
- 폰트: 'Geist Mono' 또는 'Inter' 사용, 자간을 좁게(tracking-tighter) 설정해 묵직함 부여.
- 애니메이션: framer-motion을 사용해 도트들이 성단처럼 모여드는 웅장한 효과와 숫자 슬롯머신(Ticker) 효과 구현.
- 모바일 최적화: 100dvh 단위를 사용해 하단 바 이슈 해결.

[4. 공유 및 배포 환경]
- 동적 OG 이미지: `/[domain_key]/opengraph-image.tsx`를 통해 사용자별 "남은 인생 %" 그래픽 실시간 생성.
- Web Share API: 고유 URL과 감성적인 카피를 공유하는 기능.

이상의 요구사항을 시니어 프론트엔드 개발자 수준의 클린 코드로 작성하고, 특히 '웅장한 연출'에 집중해줘.
